LCGE v1.3 Lattice Output Directory

This directory contains the discovered behavioral coordinate system.

Files generated by the pipeline:
  runs.jsonl           — Raw LLM response records (one JSON per line)
  embeddings.npz       — TF-IDF embedding matrix + run_id mapping
  distance_matrix.npz  — Pairwise cosine distance matrix
  record_index.json    — Minimal index: run_id → prompt_id, strategy, rep, row_index
  behavioral_space.json — The discovered coordinate system

behavioral_space.json structure:
  {
    "method": "PCA on TF-IDF response embeddings",
    "parameters": { "n_records", "n_features", "n_components", "empty_responses" },
    "principal_components": [
      { "index", "variance_ratio", "cumulative_variance", "eigenvalue" }
    ],
    "distance_matrix_summary": { "n_samples", "mean_distance", "std_distance", "min_distance", "max_distance" },
    "strategy_displacements": {
      "<strategy>": { "n_samples", "centroid", "displacement_from_global" }
    },
    "projections": {
      "<run_id>": [x1, x2, x3, x4, ...]
    }
  }

NO classification. NO instability types. NO triggers. NO scores.
Only empirical geometry of response space.
